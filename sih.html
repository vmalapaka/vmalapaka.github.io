Smart India Hackathon: Software Edition 2019 Winner

    Problem Statement: To develop a responsive web portal that helps startups to lodge a grievance either by a smartphone or computer to officers who handle grievances in DIPP.

    Proposed Solution: A system that allows startups to submit a grievance by specifying details like category, department, associated documents etc. regarding the issue. The system then generates a ticket and unique id to view and track the status of submitted grievance. The system also ensures that the grievances to be submitted only after successful authentication, it prevents submitting multiple grievances that have high document similarities by the same user and maintains multiple submission thresholds. Time to time notifications regarding the status and action follow up of lodged grievance are sent to the user as an Email/SMS.

    Key features of the Web Application developed:

    • Role based login implemented for 3 types of users, using intricate database deisgn in MongoDB.

    • A feedback mechanism is implemented in the system for users to report the quality of resolution. Statistics of submitted grievances and detailed information about the department are shown in the portal using dashboards.

    • A grievance analysis dashboard is implemented in the system for real-time grievance monitoring and provides category, division and region wise analysis. This dashboard is shown to the top level of the escalation matrix i.e., to higher officials of the ministry for effective monitoring of grievance redressal.

    Skills: React, REST API, MongoDB, Heroku, Threading and Networks concepts.
